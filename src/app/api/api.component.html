API utilisée ici pour le test

https://gorest.co.in

> Voir la documentation de l'API
  • Récupérer la liste des utilisateurs
    GET /users
  • Récupérer un utilisateur par son id
    GET /users/:id
  • Effacer un utilisateur
    DELETE /users/:id
  • Créer un nouvel utilisateur
    POST /users
Ces requests API sont visibles dans l'application dans src/app/services/user.service.ts

Rappel : HttpClient renvoie un Observable.
Le choix technique est de transformer ces requêtes en Promise, car ces requêtes n'ont pas vocation à être observées, mais juste exécutées une fois à la demande d'un component, et charger le résultat dans un Subject.
